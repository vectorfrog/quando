# Quando - Taskwarrior Date Parser

> 1/1 projects complete
> Started: 2026-01-01

## Overview

Quando is an Elixir library that parses Taskwarrior-style date expressions using NimbleParsec. This represents a pivot from natural language parsing to a more structured, CLI-friendly syntax used by Taskwarrior.
